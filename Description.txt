Churn Prediction Dashboard 🔮
Predict and prevent customer churn in e-commerce using Machine Learning!

This dashboard allows users to:

Predict churn for a single customer

Upload a batch of customers and analyze churn probability

Visualize churn vs stay distribution

Download batch predictions as CSV
